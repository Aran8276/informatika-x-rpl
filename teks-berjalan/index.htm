Pengertian HTML

HTML adalah singkatan dari Hypertext Markup Language. Ini adalah bahasa markup standar untuk membuat halaman web. HTML menggunakan simbol tertentu yang disebut tag, yang akan diterjemahkan oleh browser menjadi halaman web. Dengan tag-tag ini, Anda dapat membangun struktur halaman, menambahkan konten seperti teks dan gambar, serta memberikan format dan tata letak pada website.

HTML (Hypertext Markup Language) merupakan fondasi dasar dari sebuah halaman web. Bahasa ini memungkinkan kita untuk membuat dan strukturisasi konten pada internet. Setiap elemen dalam HTML dikenal sebagai tag, yang memberikan instruksi kepada web browser tentang cara menampilkan konten. Misalnya, tag p digunakan untuk paragraf, a untuk hyperlink, dan img untuk gambar. Dengan menggunakan HTML, pengembang web dapat mengatur teks, gambar, dan elemen lainnya ke dalam format yang terstruktur dan mudah diakses oleh pengguna di seluruh dunia.

Jenis-Jenis Tag HTML & Contoh Umumnya

Elemen HTML terdiri dari tag pembuka, konten, dan tag penutup. Misalnya, h1 Konten Heading h1 di mana h1 adalah tag pembuka, “Konten Heading” adalah konten, dan h1 adalah tag penutup. Ada juga elemen kosong yang tidak memiliki konten dan tag penutup, seperti br untuk baris baru. Elemen-elemen ini digunakan untuk memformat dan mengatur konten pada halaman web
